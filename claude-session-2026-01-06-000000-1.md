### Session-Saver Agent Specifications

**Filename Pattern:** `claude-session-YYYY-MM-DD-HHmmss-N.md`
- YYYY-MM-DD: Current date
- HHmmss: Current time in 24-hour format
- N: Sequential number to prevent collisions

**Required File Header:**
```markdown

**File:** /absolute/path/to/claude-session-YYYY-MM-DD-HHmmss-N.md
```
